# 电子书阅读器

一个功能丰富、界面美观的Web电子书阅读器，支持多种阅读模式和个性化设置。

## ✨ 主要功能

### 📝 笔记与高亮
- **文本高亮**：支持对重要内容进行高亮标记
- **笔记功能**：添加个人笔记和批注
- **对话高亮**：自动识别并高亮小说中的对话内容

### 阅读设置
- **字体大小**：通过右侧功能栏调整
- **行高**：自定义文本行间距
- **段落间距**：调整段落之间的距离
- **翻页模式**：切换左右翻页或滚动翻页

## 🚀 快速开始

1. 浏览器访问 https://cwj267.github.io 即可使用
2. 点击"选择文件"按钮，上传TXT格式的电子书
3. 开始阅读！

## 📋 使用说明

### 文件上传
- 支持TXT格式的电子书
- 自动检测文件编码
- 支持批量选择文件
