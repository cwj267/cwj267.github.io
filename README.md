# 电子书阅读器

一个功能丰富、界面美观的Web电子书阅读器，支持多种阅读模式和个性化设置。

## ✨ 主要功能

### 📖 阅读体验
- **多种翻页模式**：支持传统左右翻页和现代滚动翻页
- **章节无缝加载**：滚动阅读时自动加载前后章节，提供流畅的阅读体验
- **自定义字体设置**：可调整字体大小、行高和字重
- **段落间距控制**：灵活设置段落之间的间距

### 🎨 主题与外观
- **三种主题模式**：浅色主题、深色主题、护眼模式
- **跟随系统主题**：自动适配操作系统的明暗主题设置
- **动态状态栏**：Android/iOS/Windows状态栏颜色随主题自动变化
- **响应式设计**：完美适配桌面端和移动端设备

### 🔍 搜索与导航
- **全文搜索**：快速查找书中内容
- **章节跳转**：直接跳转到指定章节
- **章节列表**：清晰的章节导航界面

### 📝 笔记与高亮
- **文本高亮**：支持对重要内容进行高亮标记
- **笔记功能**：添加个人笔记和批注
- **对话高亮**：自动识别并高亮小说中的对话内容

### 🛠️ 高级功能
- **字符编码自动检测**：支持各种编码格式的TXT文件
- **性能优化**：使用DocumentFragment优化DOM操作
- **本地存储**：自动保存阅读进度和设置偏好

## 🚀 快速开始

1. 下载阅读器HTML文件
2. 用现代浏览器打开文件
3. 点击"选择文件"按钮，上传TXT格式的电子书
4. 开始阅读！

## 📋 使用说明

### 文件上传
- 支持TXT格式的电子书
- 自动检测文件编码
- 支持批量选择文件

### 阅读设置
- **字体大小**：通过右侧功能栏调整
- **行高**：自定义文本行间距
- **段落间距**：调整段落之间的距离
- **翻页模式**：切换左右翻页或滚动翻页

### 主题切换
- 点击右侧功能栏的主题选项
- 选择"跟随系统"自动适配系统主题
- 手动选择浅色、深色或护眼模式

### 章节导航
- 使用底部导航栏的"上一章"/"下一章"按钮
- 在章节列表中直接点击章节标题跳转
- 使用章节跳转输入框输入章节号

## 🛠️ 技术特点

### 前端技术栈
- **HTML5**：语义化标签，响应式设计
- **CSS3**：Flexbox布局，CSS变量，媒体查询
- **JavaScript**：ES6+特性，DOM操作，异步加载

### 核心功能实现
- **主题系统**：基于CSS变量的动态主题切换
- **章节渲染**：使用DocumentFragment优化性能
- **无缝滚动**：IntersectionObserver API实现章节预加载
- **本地存储**：localStorage保存阅读进度和设置
- **系统主题检测**：window.matchMedia API

### 性能优化
- **懒加载**：按需加载章节内容
- **DOM缓存**：减少重排重绘
- **事件委托**：优化事件处理
- **请求取消**：AbortController实现可取消的搜索
